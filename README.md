# Weather App by Makmakkerti
## Shows weather accroding to entered location

### Technologies used:
- NodeJS
- Express
- Postman-request
- Handlebars (HBS)
- Fetch
- Heroku

### Services used:
- Location service: https://www.mapbox.com/
- Forecast service: https://weatherstack.com/
- Widget template: https://codepen.io/takeradi/pen/ONJzEv
